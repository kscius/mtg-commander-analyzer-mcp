# Reanimator Strategy Guide

## Key principles

- **Fill graveyard** (mill, discard, self-mill) then **reanimate** large threats cheaply.
- **Recursion** and **protection** for reanimated targets.
- Graveyard **hate** in meta — run your own hate or recursion backup.

## Recommended ratios

| Role | Count | Notes |
|------|-------|-------|
| Lands | 35–37 | |

| Enablers | 10–14 | Mill, discard, loot |

| Removal | 5–7 | |
| Graveyard hate | 1–3 | Meta call |

## Core card types

- **Reanimate**: Reanimate, Animate Dead, Victimize
- **Enablers**: Faithless Looting, Entomb effects
- **Targets**: Your commander’s best fatties / utility creatures

## Anti-patterns

- **Too many** fatties without enablers.
- **Exile-heavy** meta with no graveyard recursion plan.
- **Tokens/voltron** taking slots from reanimation package.
- Reanimating **low-impact** creatures.

## Synergy packages

- **Entomb + reanimate**: one-card cheat
- **Mill + mass reanimate**: Living End style (check legality/synergy)
- **Discard outlet + recursion**: aristocrat crossover

## Ideal curve

- Low MV enablers; targets mostly MV 5+
- Interaction at 1–3 MV

## Commander impact

- Commanders that **reanimate**, **mill**, or **reward graveyard** define the package.
- Black is often primary; splash colors add targets and protection.
