# Aristocrats Strategy Guide

## Key principles

- **Sacrifice creatures** for value (drain, draw, tokens, counters).
- Need **recurring sacrifice outlets** and **fodder** (tokens, cheap creatures, recursion).
- Wins via **life drain**, **combat**, or **engine loops** — not random goodstuff.

## Recommended ratios

| Role | Count | Notes |
|------|-------|-------|
| Lands | 35–37 | |
| Ramp | 9–11 | |

| Death payoffs | 8–12 | Blood Artist, Zulaport, Pitiless Plunderer |
| Token fodder | 8–12 | Tokens, recursive creatures, cheap bodies |
| Removal | 5–7 | |
| Recursion | 4–6 | Reanimate fodder you sacrifice |
| Protection | 2–4 | Protect engines, not every 1/1 |

## Core card types

- **Outlets**: Viscera Seer, Ashnod’s Altar, Phyrexian Altar
- **Payoffs**: Blood Artist, Cruel Celebrant, Mayhem Devil
- **Fodder**: token makers, Persist, Undying creatures

## Anti-patterns

- Outlets without **payoffs** or fodder.
- Too many **non-creature** spells with no sacrifice synergy.
- Board wipes that leave you with no plan unless you run recursion.
- Mixing full **voltron** or **spellslinger** packages.

## Synergy packages

- **Drain loop**: outlet + Blood Artist + token maker
- **Treasure sac**: Pitiless Plunderer + fodder + mana sink
- **Graveyard**: sacrifice + reanimate same creature

## Ideal curve

- Low MV fodder and outlets; payoffs at 2–4 MV
- Finishers can be “sacrifice your board” effects (e.g. Plaguecrafter-style)

## Commander impact

- Commanders that **sacrifice on ETB/attack** or **drain** define the deck.
- Orzhov/Rakdos often need fewer slots for payoffs; other colors need more explicit drain.
