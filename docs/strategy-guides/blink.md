# Blink Strategy Guide

## Key principles

- **Flicker** creatures and permanents for repeated ETB triggers.
- Value from **enters-the-battlefield** effects, not static stats alone.
- **Protection** and **bounce** help reuse the same ETB cards.

## Recommended ratios

| Role | Count | Notes |
|------|-------|-------|
| Lands | 35–37 | |
| Ramp | 9–11 | |
| ETB payoffs | 12–18 | High-impact ETB creatures and permanents |
| Blink spells | 10–14 | Instant-speed blink is premium |
| Removal | 5–7 | |
| Draw | 8–10 | ETB draw or blink draw engines |
| Protection | 3–5 | Save key ETB creatures |

## Core card types

- **Blink**: Ephemerate, Ghostly Flicker, Displace, Eerie Interlude
- **ETB**: Risen Reef, Mulldrifter, Solemn Simulacrum (meta-dependent)
- **Engines**: Panharmonicon, Yarok-style static doublers

## Anti-patterns

- Creatures with **weak ETB** taking blink slots.
- Too few **instant-speed** blinks (only sorcery-speed is fragile).
- **Aristocrat sacrifice** without ETB payoff overlap.
- Mass wipe without **recursion** or cheap re-ETB.

## Synergy packages

- **Ephemerate + big ETB**: cheat mana, draw, removal
- **Panharmonicon + small ETB**: incremental advantage
- **Landfall + blink land**: if in green/blue

## Ideal curve

- Many MV 2–4 ETB bodies; blink at 1–3 MV
- Avoid stacking only MV 6+ ETBs without ramp

## Commander impact

- Commanders that **blink** or **reward ETB** are the deck engine.
- If commander lacks blink, prioritize cheap blink spells and ETB creatures that match commander colors.
