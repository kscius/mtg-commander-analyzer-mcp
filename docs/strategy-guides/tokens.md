# Tokens Strategy Guide

## Key principles

- Win by creating many creature tokens, then anthem or sacrifice for value.
- Prefer **one-sided** or **protected** board wipes; avoid symmetrical wipes without follow-up.
- Creature-based ramp and token makers over generic big spells.

## Recommended ratios

| Role | Count | Notes |
|------|-------|-------|
| Lands | 35–38 | Bracket 3 template |
| Ramp | 9–12 | Creature dorks and rock ramp |
| Token makers | 15–20 | Repeatable token generation |
| Anthems / buffs | 5–8 | Team-wide pumps |
| Sacrifice outlets | 3–5 | Optional value line |
| Mass protection | 3–5 | Save wide board |
| Removal | 5–7 | Spot + 2–4 wipes |
| Draw | 8–11 | Keep gas for go-wide turns |

## Core card types

- **Token makers**: parallel lives effects, repeatable creators
- **Anthems**: Intangible Virtue, Cathars' Crusade, Mondrak
- **Protection**: Heroic Intervention, Teferi's Protection

## Anti-patterns

- Mass creature wipes with no recovery
- Cards that sacrifice all your creatures
- Mixing unrelated voltron without token payoffs

## Synergy packages

- **Go Wide**: Anointed Procession, Doubling Season, Parallel Lives
- **Token Sacrifice**: Ashnod's Altar, Skullclamp, Impact Tremors

## Ideal curve

- MV 2–4 token makers and anthems; finishers at 4–6 MV
- Avoid stacking only high-CMV spells without ramp

## Commander impact

- Commanders that **create tokens** or **reward wide boards** reduce redundant makers.
- Color identity sets whether sacrifice, enchantment, or combat finishers fit best.
