# Lands Matter Strategy Guide

## Key principles

- **Play extra lands**, **landfall**, and **lands in hand** as resources.
- Mana base is **the deck** — prioritize land count and land types.
- Win with **large land count**, **land animation**, or **landfall payoffs**.

## Recommended ratios

| Role | Count | Notes |
|------|-------|-------|
| Lands | 38–45+ | Above template minimum is normal |
| Ramp | 6–10 | Land ramp > rocks often |
| Landfall payoffs | 10–16 | |
| Land enablers | 8–12 | Extra land drops, bounce lands |
| Removal | 4–6 | |
| Draw | 8–10 | Land-based draw |

## Core card types

- **Extra land**: Exploration, Azusa, Oracle of Mul Daya
- **Landfall**: Tireless Tracker, Avenger of Zendikar
- **Payoffs**: Obelisk of Green, Rampaging Baloths

## Anti-patterns

- **Too few lands** for the strategy.
- **Mass land destruction** (banned in Bracket 3).
- **Non-land** ramp that does not support landfall.
- Typical 35-land midrange manabase.

## Synergy packages

- **Extra land + landfall**: Azusa + Retreat to Coralhelm
- **Discarded lands**: land draw + reanimation of lands (if applicable)
- **Combo lands**: Thespian’s Stage + … (check Bracket 3 combo rules)

## Ideal curve

- Lower spell MV; lands are “free” plays
- Payoffs at 3–5 MV landfall

## Commander impact

- Green commanders often **enable extra lands**; build around commander land drops.
- Non-green partners need land color fixing and payoffs in those colors.
