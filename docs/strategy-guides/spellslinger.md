# Spellslinger Strategy Guide

## Key principles

- **Instants and sorceries** matter; creatures are support.
- **Copy**, **discount**, and **cast triggers** generate advantage.
- **Interaction density** is naturally high — balance with win conditions.

## Recommended ratios

| Role | Count | Notes |
|------|-------|-------|
| Lands | 35–36 | |
| Ramp | 9–11 | Rocks + cost reducers |
| Instants/sorceries | 35–45 | Mix draw, removal, payoff |
| Creatures | 8–15 | Magecraft, reduction creatures |
| Copy / storm payoffs | 4–8 | |
| Win conditions | 2–4 | Burn, extra turns (Bracket cap) |

## Core card types

- **Payoffs**: Storm-Kiln Artist, Archmage Emeritus, Guttersnipe
- **Copy**: Twincast, Swarm Intelligence
- **Draw**: instant-speed draw, wheel effects (meta-dependent)

## Anti-patterns

- **Creature-heavy** midrange with no spell synergy.
- Too many **high MV** sorceries without ramp.
- **Extra turns** over Bracket 3 cap.
- Mixing **voltron** as primary plan.

## Synergy packages

- **Cantrip + payoff**: cheap spells + Magecraft
- **Copy big spell**: Fork + game-ending sorcery
- **Commander cast triggers**: spells matter commander + ritual effects

## Ideal curve

- Many MV 1–3 spells; finishers at 4–6 MV
- Keep avg MV ≤ template target (~3.2)

## Commander impact

- Izzet-style commanders reward **casting two spells** or **instants**.
- Adjust creature count down if commander is non-creature focused.
