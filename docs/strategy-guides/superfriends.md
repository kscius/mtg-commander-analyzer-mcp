# Superfriends (Planeswalkers) Strategy Guide

## Key principles

- **Planeswalkers** as primary win conditions and engines.
- **Proliferate** and **protection** extend loyalty plans.
- **Choke the ground** with multiple walkers — need time and removal.

## Recommended ratios

| Role | Count | Notes |
|------|-------|-------|
| Lands | 36–38 | |
| Ramp | 10–12 | Needs consistent mana for multiple MV4+ spells |
| Planeswalkers | 8–14 | Mix draw, removal, ultimates |
| Proliferate | 4–7 | |
| Removal | 6–8 | Protect walkers |
| Board wipes | 2–3 | Reset creature boards |
| Draw | 6–8 | Some from walkers |

## Core card types

- **Walkers**: value engines in your colors
- **Proliferate**: Karn’s Bastion, Inexorable Tide
- **Protection**: Teferi’s Protection, counterspells

## Anti-patterns

- **Too few** walkers for proliferate payoffs.
- **Creature-only** deck with 2 walkers.
- **No removal** — walkers die to combat and burn.
- Extra turns over Bracket cap.

## Synergy packages

- **Proliferate + ult**: rush emblem (meta-dependent)
- **Doubling Season + walker**: check Bracket / salt
- **Stax light**: not full stax in Bracket 3 — prefer interaction

## Ideal curve

- Ramp at 2–3 MV; walkers at 3–6 MV
- Few pure combat creatures

## Commander impact

- Commanders that **proliferate** or **reduce walker cost** are core.
- Color identity often Esper, Bant, or 5-color for walker breadth.
